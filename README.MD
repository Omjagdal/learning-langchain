# 🚀 Learning LangChain

A hands-on repository to learn and experiment with **LangChain**, focusing on building real-world **LLM-powered applications** like chatbots, agents, and retrieval systems.

---

## 📌 Overview

This repository contains practical implementations and experiments using **LangChain**, a framework for building applications powered by Large Language Models (LLMs).

It is designed for:

* Beginners exploring LangChain
* Developers building AI-powered applications
* Students preparing for AI/ML roles

---

## 🧠 What is LangChain?

LangChain is a framework that helps developers build applications using LLMs by integrating:

* Prompts
* Chains
* Agents
* Memory
* External tools & APIs

---

## ⚙️ Features

* 🔹 Prompt Engineering examples
* 🔹 Chains (Sequential & Custom workflows)
* 🔹 Agents with tool usage
* 🔹 Memory integration
* 🔹 Retrieval-Augmented Generation (RAG)
* 🔹 API integrations (OpenAI / others)
* 🔹 Real-world mini projects

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Omjagdal/learning-langchain.git
cd learning-langchain
```

---

### 2️⃣ Create virtual environment

```bash
python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows
```

---

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

### 4️⃣ Set Environment Variables

Create a `.env` file:

```env
OPENAI_API_KEY=your_api_key_here
```

---

### 5️⃣ Run examples

```bash
python scripts/example.py
```

or open notebooks:

```bash
jupyter notebook
```
